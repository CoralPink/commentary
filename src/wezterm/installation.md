```admonish info title="[Install - Wez's Terminal Emulator](https://wezfurlong.org/wezterm/installation.html)"

WezTermは主要なプラットフォーム用にあらかじめビルドされていますが、オープンソースであるため、自分でビルドすることも可能です。
```
