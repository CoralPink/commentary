    Motivation behind 'copilot.lua'

    While using copilot.vim, for the first time since I started using neovim my laptop began to overheat. Additionally, I found the large chunks of ghost text moving around my code, and interfering with my existing cmp ghost text disturbing. As lua is far more efficient and makes things easier to integrate with modern plugins, this repository was created.

    copilot.vimгӮ’дҪҝгҒЈгҒҰгҒ„гӮӢгҒЁгҒҚгҖҒneovimгӮ’дҪҝгҒ„е§ӢгӮҒгҒҰгҒӢгӮүеҲқгӮҒгҒҰз§ҒгҒ®гғ©гғғгғ—гғҲгғғгғ—гҒҢгӮӘгғјгғҗгғјгғ’гғјгғҲгҒ—е§ӢгӮҒгҒҫгҒ—гҒҹгҖӮ гҒ•гӮүгҒ«гҖҒгӮҙгғјгӮ№гғҲгғҶгӮӯгӮ№гғҲгҒ®еӨ§гҒҚгҒӘеЎҠгҒҢз§ҒгҒ®гӮігғјгғүгҒ®е‘ЁгӮҠгӮ’еӢ•гҒҚеӣһгӮҠгҖҒж—ўеӯҳгҒ®cmpгӮҙгғјгӮ№гғҲгғҶгӮӯгӮ№гғҲгҒЁе№ІжёүгҒ—гҒҰгҒ„гӮӢгҒ“гҒЁгҒ«ж°—гҒҘгҒҚгҒҫгҒ—гҒҹгҖӮ luaгҒ®ж–№гҒҢгҒҜгӮӢгҒӢгҒ«еҠ№зҺҮзҡ„гҒ§гҖҒжңҖж–°гҒ®гғ—гғ©гӮ°гӮӨгғігҒЁгҒ®зөұеҗҲгҒҢе®№жҳ“гҒ«гҒӘгӮӢгҒҹгӮҒгҖҒгҒ“гҒ®гғ¬гғқгӮёгғҲгғӘгҒҢдҪңжҲҗгҒ•гӮҢгҒҫгҒ—гҒҹгҖӮ

    Setup and Configuration

    Because the copilot server takes some time to start up, it is recommend that you lazy load copilot. For example:

    copilot гӮөгғјгғҗгғјгҒҜиө·еӢ•гҒ«жҷӮй–“гҒҢгҒӢгҒӢгӮӢгҒҹгӮҒгҖҒcopilot гӮ’йҒ…е»¶гғӯгғјгғүгҒҷгӮӢгҒ“гҒЁгӮ’гҒҠеӢ§гӮҒгҒ—гҒҫгҒҷгҖӮдҫӢгҒҲгҒ°гҖҒд»ҘдёӢгҒ®гӮҲгҒҶгҒӘж„ҹгҒҳгҒ§гҒҷпјҡ

    use {
  "zbirenbaum/copilot.lua",
  cmd = "Copilot",
  event = "InsertEnter",
  config = function()
    require("copilot").setup({})
  end,
}
