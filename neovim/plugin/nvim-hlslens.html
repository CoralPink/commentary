    nvim-hlslens helps you better glance at matched information, seamlessly jump between matched instances.

    nvim-hlslens гҒҜгҖҒгғһгғғгғҒгҒ—гҒҹжғ…е ұгӮ’гӮҲгӮҠиҰӢгӮ„гҒҷгҒҸгҒ—гҖҒгғһгғғгғҒгҒ—гҒҹгӮӨгғігӮ№гӮҝгғігӮ№й–“гӮ’гӮ·гғјгғ гғ¬гӮ№гҒ«гӮёгғЈгғігғ—гҒ§гҒҚгӮӢгӮҲгҒҶгҒ«гҒ—гҒҫгҒҷгҖӮ

    :h quickfix

    Vim has a special mode to speedup the edit-compile-edit cycle.  This is
inspired by the quickfix option of the Manx's Aztec C compiler on the Amiga.
The idea is to save the error messages from the compiler in a file and use Vim
to jump to the errors one by one.  You can examine each problem and fix it,
without having to remember all the error messages.

VimгҒ«гҒҜгҖҒз·ЁйӣҶ-гӮігғігғ‘гӮӨгғ«-з·ЁйӣҶгҒ®гӮөгӮӨгӮҜгғ«гӮ’й«ҳйҖҹеҢ–гҒҷгӮӢгҒҹгӮҒгҒ®зү№еҲҘгҒӘгғўгғјгғүгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ
гҒ“гӮҢгҒҜ Amiga гҒ® Manx гҒ® Aztec C гӮігғігғ‘гӮӨгғ©гҒ® quickfix гӮӘгғ—гӮ·гғ§гғігҒ«гғ’гғігғҲгӮ’еҫ—гҒҹгӮӮгҒ®гҒ§гҒҷгҖӮ
гҒ“гӮҢгҒҜгҖҒгӮігғігғ‘гӮӨгғ©гҒӢгӮүгҒ®гӮЁгғ©гғјгғЎгғғгӮ»гғјгӮёгӮ’гғ•гӮЎгӮӨгғ«гҒ«дҝқеӯҳгҒ—гҒҰгҒҠгҒҚгҖҒVim гӮ’дҪҝгҒЈгҒҰдёҖгҒӨгҒҡгҒӨгӮЁгғ©гғјгҒ«гӮёгғЈгғігғ—гҒҷгӮӢгҒЁгҒ„гҒҶгӮӮгҒ®гҒ§гҒҷгҖӮ
гӮЁгғ©гғјгғЎгғғгӮ»гғјгӮёгӮ’гҒҷгҒ№гҒҰиҰҡгҒҲгҒҰгҒҠгҒӢгҒӘгҒҸгҒҰгӮӮгҖҒгҒқгӮҢгҒһгӮҢгҒ®е•ҸйЎҢгӮ’иӘҝгҒ№гҒҰдҝ®жӯЈгҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

In Vim the quickfix commands are used more generally to find a list of
positions in files.  For example, |:vimgrep| finds pattern matches.  You can
use the positions in a script with the |getqflist()| function.  Thus you can
do a lot more than the edit/compile/fix cycle!

Vim гҒ§гҒҜгҖҒquickfix гӮігғһгғігғүгҒҜгӮҲгӮҠдёҖиҲ¬зҡ„гҒ«гҖҒгғ•гӮЎгӮӨгғ«еҶ…гҒ®дҪҚзҪ®гҒ®гғӘгӮ№гғҲгӮ’иҰӢгҒӨгҒ‘гӮӢгҒҹгӮҒгҒ«дҪҝз”ЁгҒ•гӮҢгҒҫгҒҷгҖӮ

гҒ“гҒ®дҪҚзҪ®гҒҜ |getqflist()| й–ўж•°гӮ’дҪҝгҒЈгҒҰгӮ№гӮҜгғӘгғ—гғҲгҒ§дҪҝз”ЁгҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ
гҒ“гҒ®гӮҲгҒҶгҒ«гҖҒз·ЁйӣҶ/гӮігғігғ‘гӮӨгғ«/дҝ®жӯЈгҒ®гӮөгӮӨгӮҜгғ«гӮҲгӮҠгӮӮеӨҡгҒҸгҒ®гҒ“гҒЁгҒҢгҒ§гҒҚгӮӢгҒ®гҒ§гҒҷ!
