    SEPARATORS

lualine defines two kinds of separators:

lualine гҒҜ2зЁ®йЎһгҒ®гӮ»гғ‘гғ¬гғјгӮҝгӮ’е®ҡзҫ©гҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

- `section_separators`   - separators between sections
                           гӮ»гӮҜгӮ·гғ§гғій–“гҒ®гӮ»гғ‘гғ¬гғјгӮҝгҖӮ

- `component_separators` - separators between the different components in sections
                           гӮ»гӮҜгӮ·гғ§гғіеҶ…гҒ®з•°гҒӘгӮӢгӮігғігғқгғјгғҚгғігғҲгӮ’еҢәеҲҮгӮӢгӮ»гғ‘гғ¬гғјгӮҝ

      options = {
    section_separators = { left = 'оӮҙ', right = 'оӮ¶' },
    component_separators = { left = 'оӮө', right = 'оӮ·' }
  }

    Here, left refers to the left-most sections (a, b, c), and right refers to the
right-most sections (x, y, z).

гҒ“гҒ“гҒ§гҖҒleft гҒҜе·Ұз«ҜгҒ®гӮ»гӮҜгӮ·гғ§гғі(a, b, c)гӮ’гҖҒright гҒҜеҸіз«ҜгҒ®гӮ»гӮҜгӮ·гғ§гғі(x, y, z)гӮ’ж„Ҹе‘ігҒ—гҒҫгҒҷгҖӮ

    гҒҫгҒӮзҙ°гҒӢгҒ„гҒ“гҒЁгҒҜеҫҢгҒ«гҒ—гҒҫгҒ—гӮҮгҒҶгҖӮгҒ“гӮҢгҒҜгӮӮгҒҶдҫӢзӨәгҒ•гӮҢгҒҰгҒ„гӮӢйҖҡгӮҠгҒ«гӮ„гҒЈгҒЎгӮғгҒ„гҒҫгҒҷрҹҳҶ
