    nvim-scrollbar

    Extensible Neovim Scrollbar

    拡張可能なNeovimスクロールバー

    Requirements

    Neovim >= 0.5.1

    nvim-hlslens (optional)

    gitsigns.nvim (optional)

    nvim-hlslensもgitsigns.nvimも、とっても見覚えあるやつですね😉

    両方ともoptionalということですが、これはぜひ積極的に取り入れていきましょう❗

    Note

    というか、最初から取り入れるつもりで進めてたんですけどね😸

    Installation

    これはもう本当に簡単です。1

    extensions/nvim-scrollbar.lua

    require('scrollbar').setup()

require('scrollbar.handlers.search').setup()
require("scrollbar.handlers.gitsigns").setup()

    Note

    もし細かく設定したい場合は config にある Defaults を参照すると良いです。

    このコードでは、

    require('scrollbar.handlers.search').setup() -- これは nvim-hlslens

    require("scrollbar.handlers.gitsigns").setup()

    ...で、それぞれのプラグインを必要とします。pakcerにも教えといてあげましょう🫶

    extensions/init.lua

      use {
    'petertriho/nvim-scrollbar',
    config = function() require 'extensions.nvim-scrollbar' end,
    requires = {
      'kevinhwang91/nvim-hlslens', 'lewis6991/gitsigns.nvim',
    },
  }
